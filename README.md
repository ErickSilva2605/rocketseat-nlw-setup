<h2 align="center">Primeiros Passos 🚀</h2>

   ```
   - Clone o projeto 

   $ git clone https://github.com/ErickSilva2605/rocketseat-nlw-setup.git
   $ cd nlw-setup-ignite
   ```

   ### Web
   ```
   # Instalar as dependências
   $ cd web
   $ npm install

   # Inicia o projeto
   $ npm start
   ```

   ### Server
   ```
   # Instalar as dependências
   $ cd server
   $ npm install

   # Inicia o projeto
   $ npx prisma migrate deploy
   $ npm run dev
   ```

   ### Mobile
   ```
   # Instalar as dependências
   $ cd mobile
   $ npm install

   # Inicia o projeto
   $ npx expo start
   ```
